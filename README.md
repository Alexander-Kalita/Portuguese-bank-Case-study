# Portuguese bank Case study

This repository contains code and files related to a case study on a Portuguese bank dataset. 
The case study focuses on analyzing customer data and building classification models for predicting customer subscriptions to term deposits.

## Files

- `Portuguese bank case study.ipynb` [
JupyterLab containing the code, analysis, and explanations for the case study.
- `Portuguese-bank-code`: Directory containing the necessary code files and modules for the case study.
